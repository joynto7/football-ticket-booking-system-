# Football Ticket Booking System

This repository contains the design and SQL queries for a simple football ticket booking system.

## What’s included

- Database schema for users, matches, and bookings
- Relationship logic for users and bookings
- Example SQL query scenarios for match search, booking lookup, and user reporting

## Key features

- Users can be football fans or ticket managers
- Matches include fixture info, tournament category, price, and availability
- Bookings connect users with matches and track payment and cost details

## Tables

1. `Users`
2. `Matches`
3. `Bookings`

This README is meant to keep the project simple, clear, and easy to review.
